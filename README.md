# **Flutter** & **Go** Logistics App

A logistics app for both users and companies to track, manage, and coordinate deliveries.

---

### 🚀 Features

#### 👤 Users Can:
- Order shipping services
- Track their orders

#### 🏢 Companies Can:
- Manage employees
- Manage orders
- Manage ships & ports
